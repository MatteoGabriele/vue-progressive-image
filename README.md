# vue-progressive-image

Vue progressive image loading plugin

![alt tag](https://raw.githubusercontent.com/MatteoGabriele/vue-progressive-image/master/example.gif)

## Requirements

Vue ^3.0.0

## Install

```bash
npm install vue-progressive-image
```
